PL/R - Win32 Notes

PL/R 8.3.0.5-beta

PL/R 8.2.0.7

PL/R 0.6.2.8

- Be sure to read the installation section of the documentation.
- Current sources will not compile under Win32 due to a missing header file -- specifically Rinterface.h. You can work around this issue by obtaining a copy of the header file from R sources for your version of R, and placing it in $R_HOME/include. This workaround will be eliminated in the next release.
- When building you may see "make: pkg-config: Command not found". PL/R should still build and work fine, as long as R_HOME is properly defined. However pkg-config can be installed and will assist in setting R_HOME under the right circumstances. Follow the instructions here to get setup for pkg-config.
